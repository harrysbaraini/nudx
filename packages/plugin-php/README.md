@nudx/plugin-php
================

Add PHP to Nudx CLI

# Install

Add this plugin to Nudx by running:

```
nudx plugins:install @nudx/plugin-php
```

# Options

| option  | type     | default   | info                        | choices             |
|---------|----------|-----------|-----------------------------|---------------------|
| version | string   | "8.2"     |                             | "8.0", "8.1", "8.2" |
|         |          |           |                             |                     |
