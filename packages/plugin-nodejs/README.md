@nudx/plugin-nodejs
===================

Add Node.js to Nudx CLI

# Install

Add this plugin to Nudx by running:

```
nudx plugins:install @nudx/plugin-nodejs
```

# Options

| option  | type     | default   | info                        | choices                |
|---------|----------|-----------|-----------------------------|------------------------|
| version | string   | 18        |                             | "19", "18", "16", "14" |
|         |          |           |                             |                        |
