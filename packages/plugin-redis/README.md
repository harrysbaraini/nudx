@nudx/plugin-redis
==================

Add Redis to Nudx CLI

# Install

Add this plugin to Nudx by running:

```
nudx plugins:install @nudx/plugin-redis
```

# Options

| option  | type     | default   | info                        | choices |
|---------|----------|-----------|-----------------------------|---------|
| port    | string   | 6379      |                             |         |
|         |          |           |                             |         |
