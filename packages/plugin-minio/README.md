@nudx/plugin-minio
=================

Add Minio to Nudx CLI

# Install

Add this plugin to Nudx by running:

```
nudx plugins:install @nudx/plugin-minio
```

# Options

| option  | type     | default   | info                        | choices |
|---------|----------|-----------|-----------------------------|---------|
| port    | string   | 9000      |                             |         |
| buckets | string[] | ["files"] | A list of buckets to create |         |
|         |          |           |                             |         |
