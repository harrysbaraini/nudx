@nudx/plugin-elasticsearch
==========================

Add ElasticSearch to Nudx CLI

# Install

Add this plugin to Nudx by running:

```
nudx plugins:install @nudx/plugin-elasticsearch
```

# Options

| option  | type     | default   | info                        | choices  |
|---------|----------|-----------|-----------------------------|----------|
| port    | string   | 9200      |                             |          |
| version | string   | "7"       |                             | "7", "6" |
|         |          |           |                             |          |
